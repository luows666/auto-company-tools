# Auto Tools - 100% Offline Developer Toolkit

A privacy-first offline toolkit with 7 essential utilities. Works directly in your browser, no internet required.

[Live Demo](https://luows666.github.io/auto-company-tools/)

## Features

- **Password Generator** - Customizable length and character sets
- **JSON Formatter** - Format, validate, minify JSON
- **Base64 Encoder/Decoder** - Encode and decode Base64
- **URL Encoder/Decoder** - URL-safe encoding
- **Hash Generator** - SHA-1 and SHA-256
- **Unit Converter** - Length, temperature, weight
- **Budget Tracker** - Personal finance with localStorage
- **Timer & Countdown** - Stopwatch and alarm
- **Color Picker** - HEX, RGB, HSL conversion

## Why This?

1. **100% Offline** - No internet required, works in airplane mode
2. **Privacy-First** - No data leaves your browser
3. **Single File** - No installation, just copy and run
4. **Zero Dependencies** - Pure vanilla JavaScript

## For Developers

### Deployment to GitHub Pages
```bash
# If using this repo
git clone https://github.com/luows666/auto-tools.git
cd auto-tools
# The index.html can be deployed to any static hosting
```

### Customization
All tools are in a single HTML file. Modify directly:
- Add new tools in the tool-sections
- Add JavaScript functions in the script tag
- Style with CSS in the style tag

## Tech Stack

- Pure HTML/CSS/JavaScript
- No frameworks
- No external dependencies
- localStorage for data persistence
- Web Crypto API for hashing
